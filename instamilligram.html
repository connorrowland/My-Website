InstaMilligram

What it is

InstaMilligram is an photo sharing site intended to replicate the experience of using Instagram. From a developer standpoint, InstaMilligram is a tool I can use to build more projects in the future with, as it is a fully functioning Rails CRUD app with login capability.

Skills Used

Ruby

Rails

JQuery/AJAX

Heroku

AWS S3

Why I Made It

Beyond aiming to replicate Instagram, my intentions with InstaMilligram were to sharpen my Rails and AJAX/JQuery skills, and build an application with RESTful considerations in mind that so I could use it as a solid foundation for other projects. I encountered a few bumps along the way in building InstaMilligram as unexpected situations came about. But these challenges ended up teaching me so much more than I expected, and also prepared me for my work at Ellevest, where I spent a good deal of time adding to and refactoring pieces of Rails products.

A Frame For Future Projects

I hope to use this project and its various functionality pieces—like loggin in and commenting—as a frame for many new projects to come. I see InstaMilligram as a blank canvas (functionally) that I can add on new ideas and projects concepts to. I find having this type of product to be an asset as an engineer because by having a functionality frame, I've already gotten over a huge hurdle in the implementation of future products.

Difficulties Along the Way

This project also shed light on many of the challenges associated with building a product from the ground up. But it was in these challenges that I truly got to know not only the product, but the technologies that support it (and do not...). Deployments in particular were one hurdle for me to overcome. Looking back to my time at Ellevest, I definitely took their well developed deployment process for granted. So when it was time for me to do this on my own with InstaMilligram, I ran into a few bumps along the way (that entailed so many "hope this works" commit messages), but eventually gained a great understanding of the Rails Asset Pipeline.

Another challenge I faced with InstaMilligram came from using Heroku for a photo sharing website without realizing that Heroku's database storage is ephemeral (*shakes head*). So I ended up learning how to connect a Rails application hosted on Heroku with AWS S3 and the world became good again!

These challenges—along with many others that I've encountered as I begin to grow into my place as a web engineer—and the way I've learned to deal with them are outlined in a blog post I've written here
